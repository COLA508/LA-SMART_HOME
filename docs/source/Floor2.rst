三、小屋二层的安装
===========================

步骤 1：窗户部分的安装
---------------------------

步骤1-1：齿轮与摆臂的安装
零件清单：亚克力齿轮、舵机包（单向摆臂）、M1.5*5mm自攻螺丝（1颗）。

安装如下图所示：
~~~~~~~~~~~~~~~~~~~~~~~~~~~
.. image:: _static/齿轮摆臂安装图.png
   :alt: 窗户齿轮
   :align: center
   :width: 400px

注意事项：
 - 齿轮的小孔（左右两个均可）与摆臂第三个小孔对齐安装，再拧紧螺丝。
 - 此处安装与后续的大门控制齿轮安装步骤及零件一致，可在此步骤一并安装。
